## Overview

This is a modern, responsive, and dark-themed personal portfolio website built to showcase my skills, experience, and projects as a Software Developer. It features smooth animations, a clean UI/UX, and a seamless browsing experience across all devices.

The portfolio serves as a central hub for my professional identity, including my CV, project case studies, and contact information.

## Features

- Modern Dark Theme: A sleek, professional dark mode design using a slate and blue color palette.
- Responsive Design: Fully responsive layout ensuring compatibility with mobile, tablet, and desktop devices.
- Smooth Animations: Integrated Framer Motion for engaging entrance animations, hover effects, and page transitions.
- Interactive UI: Dynamic navigation bar, interactive project cards, and animated buttons.
- CV Integration: Direct download link for my professional resume.

## Tech Stack

- Frontend Framework: React
- Build Tool: Vite
- Styling: Tailwind CSS
- Animations: Framer Motion
- Icons: Lucide React
